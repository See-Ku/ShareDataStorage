
# ShareDataStorage

ShareDataStorageはiOS標準のShareを経由して、いろんなデータを一時的に保管するだけのアプリです。


## 動作環境

|環境	|情報			|
|-------|---------------|
|Xcode	|7.3 (7D175)	|
|iOS	|9.0			|
|Swift	|2.2			|
|Date	|2016/4/7		|
